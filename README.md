# nofap

Nofap counter 

## Usage 
To start an interactive mode and list all available commands do:
```sh
$ ./nofap.py
[NoFap]>help
```
* `nr, new record`           Set a new record 
* `nd, nofap days`           Number of days from last fap 
* `nc, nofap calendar`       Open nofap data file with less
